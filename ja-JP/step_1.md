## はじめに

Pythonを使って、信号灯のLEDを制御したり、決まった順番に(シーケンス)点灯させるコーディングの方法を学びます。

\--- collapse \---

* * *

## title: 必要なもの

- Raspberry Pi コンピュータ
- LED: 
    - pi-stop 信号灯ボード
    - またはLEDと抵抗器各3本

Muをインストールする必要があります。 ターミナルウィンドウを開き、次のコマンドをタイプします。

```bash
sudo apt install mu-editor
```

\--- /collapse \---

\--- collapse \---

* * *

## title: 学ぶこと

- Raspberry PiのGPIOピンをPythonでコントロールすること
- Raspberry PiでLEDを使うこと
